# Deep Learning (Oyun Dünyası Versiyonu)

Derin öğrenme alanında, egzersizlerin oyun dünyası veri setleriyle hazırlanmış kişisel alıştırma
versiyonlarını içeren klasör.

## Tamamlanan Projeler

| Proje | Konu | Veri Seti |
|-------|------|-----------|
| [CNN/01-fashion-mnist-cnn](CNN/01-fashion-mnist-cnn) | Görüntü sınıflandırma (CNN) | Kaggle: `fronkongames/steam-games-dataset` (kapak görselleri) |
| [CNN/02-goruntu-on-isleme](CNN/02-goruntu-on-isleme) | Augmentation, PCA, t-SNE | Sentetik oyun ikonları (dış veri seti gerektirmiyor) |
| [CNN/03-opencv](CNN/03-opencv) | Klasik görüntü işleme (OpenCV) | Kaggle: `fronkongames/steam-games-dataset` (kapak görseli) |

## Kullanılan Teknolojiler

- TensorFlow / Keras
- OpenCV, Pillow
- scikit-learn (PCA, t-SNE, MLPClassifier)
- kagglehub, pandas, requests
- NumPy, Matplotlib
